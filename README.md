# solder
i made a circuit board fidget toy. it has a a potentiometer that turns on an LED, a button that toggles an LED, and a photoresistor that turns on a motor. 

i used: 
3 resistors
2 LEDs
2 NPN transistors
1 mini motor disk
1 potentiometer
1 photoresistor
1 push button
1 battery holder
